# Getting Started
This documentation covers the steps to integrating your apps and website with **inTrack**. We’ll also discuss a few fundamentals of how the [dashboard](https://dash.intrack.ir) works in tandem with your business to provide a 360° view of your users and engage them with contextually personalized messages.

Let's get you started!
## Introduction
**inTrack** is an all-in-one marketing automation solution designed to empower businesses. It enables organizations to establish an automated, personalized, and highly efficient line of communication with their customers, driving growth and enhancing customer engagement.

## Step 1: Platform Integration
Connect your platforms with **inTrack** through an SDK Integration. This can be done through any of the following methods:
### SDK
- [Web SDK Integration](web/web-getting-started.md)
- [Android SDK Integration](android/android-getting-started.md)
- [iOS SDK Integration](iOS/ios-getting-started.md)
- [React Native SDK Integration](react-native/react-native-getting-started.md)
- [Flutter SDK Integration](flutter/flutter-getting-started.md)

### API
- [REST API Integration](rest-api/rest-api-getting-started.md)
- [Transactional Campaign API](rest-api/transactional-campaign-api.md)

Once you've connected all your platforms, we will automatically start tracking specific pre-defined data points as System Events.

## Step 2: Track User Data & Behavioral Data
You can track and pass custom user data and behavioral data ([Custom Events](events#custom-events))to your [dashboard](https://dash.intrack.ir) for all your anonymous and identified users.

## Step 3: Channel Setup
Configure Push, In-app, On-site, to start engaging users. Doing so will also enable you to build cross-channel user experiences through Journeys & automate product communication through Relays.

## Step 4: Start Engaging Users!
Head over to our User Documentation to understand how you can leverage **inTrack**'s channels and analytical capabilities to deliver impactful user experiences.
