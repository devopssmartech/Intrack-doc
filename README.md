# inTrack Documentation (Markdown)

This repository contains **plain Markdown** exports of the [inTrack documentation](https://docs.intrack.ir) (version **3.0.0+**).

It is updated automatically when the main docs repo changes on GitLab.

## For AI tools

- Browse `llms.txt` for a page index.
- Each `.md` file includes a `docs_url` in the frontmatter linking to the live page.
- Content is stripped of Docusaurus-specific MDX (tabs, imports, React components).
